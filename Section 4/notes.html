Intermediate HTML

Unit Objectives

Pure Html unit with 2 major topics

  • Tables
    • Understand/Write valid HTML tables
    • Table Exercise
  • Forms
    • Write valid forms using tags for:
      • form
      • input
      • select
      • label
  • Complex form exercise

Tables

  • table - Starts table element
  • tr - Table row element
  • td - Table data element
  • th - Table header element
  • thead - specifies table header section
  • tbody - specifies table body section

Example

Name Age Breed
Rusty 2 Mutt
Wyatt 13 Golden

Forms

  • form tag
    • action - the URL to send form data to
    • method - the type of HTTP request
  • input tag
    • name - attribute for giving form input a "variable" name
    • text
    • date
    • color
    • file
    • checkbox
  • label tag
  • simple validations
