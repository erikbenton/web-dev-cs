Intro to CSS

  • Style tags
    • <style type="text/css">
  • Cascading Style Sheets (CSS) - "adjectives" of a website
    • <link rel="stylesheet" href="app.css">
  • General rules
    • selector {
    • property: value;
    • }
  • Colors
    • generic names: red, lightgray, etc.
    • Hexadecimal - #FFAA33
    • rgb() - rgb(255, 128, 0)
    • rgba() - "a" stands for alpha, or transparency
